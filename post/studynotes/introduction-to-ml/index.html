Introduction to Machine Learning

Jan 14, 2023
2 minute read

Machine Learning (ML)

Supervised machine learning - combine input to produce useful predictions on new data Unsupervised machine learning

Key Terminology

lable: the variable we are predicting (y) ex: house price, sales

feature: input variable (x) ex: sender’s email address, time, words

  • Useful features are quantifiable.
  • We train the model with labeled examples and predict the lable on unlabled data.

model: system that defines the relationship between features and lable training: creating or learning the model inference: appling the trained model to unlabled examples => use the model to predict y

regression - for continuous values ex: what is the price? classification - for discrete values ex: is it spam or not spam?

L{2} loss

= squared loss sum of (y - prediction(x))^2

Linear Regression

By convention in machine learning, equation for a model is slightly different:

y′=b+w1x1

where:

  • y′ is the predicted label (a desired output).
  • b is the bias (the y-intercept), sometimes referred to as w0.
  • w1 is the weight of feature 1. Weight is the same concept as the “slope” m in the traditional equation of a line.
  • x1 is a feature (a known input).

Training and Loss

GOAL of training is minimize loss.

empirical risk minimization: the process which tries to minimize loss by examining a lot of examples.

Mean square error (MSE) is the average squared loss per example over the whole dataset. To calculate MSE, sum up all the squared losses for individual examples and then divide by the number of examples:

MSE=1N∑(x,y)∈D(y−prediction(x))2

where:

  • (x,y) is an example in which
    • x is the set of features (ex: minute, age, gender) that the model uses to make predictions.
    • y is the example’s label (ex: example, temperature).
  • prediction(x) is a function of the weights and bias in combination with the set of features x.
  • D is a data set containing many labeled examples, which are (x,y) pairs.
  • N is the number of examples in D.
